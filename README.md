# AcmeSecure - Professional Security Solutions 🛡️

A high-performance, responsive web platform built for **AcmeSecure**, a leading security provider in the United Kingdom. This project showcases modern web standards, focusing on accessibility, security, and branding.

## ✨ Key Features
* **Service Listings:** Detailed sections for CCTV, Manned Guarding, Event Security, and Dog Security.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
* **Interactive UI:** Custom-built navigation with Lucide icons and FontAwesome social integrations.
* **Branding:** Consistent color theory and typography (Montserrat) as per client requirements.

## 🛠️ Tech Stack
* **Frontend:** React.js, TypeScript
* **Styling:** Tailwind CSS (with custom branding configuration)
* **Icons:** Lucide React & FontAwesome Brands
* **Routing:** React Router DOM
